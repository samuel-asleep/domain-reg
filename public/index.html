InfinityFree Automation API

Manage your InfinityFree hosting accounts, domains, and DNS records programmatically

Welcome! This API allows you to automate InfinityFree hosting operations. Use the tabs above to explore different features.

Verify Authentication

Test that your InfinityFree cookies are properly configured and working.

View Accounts

Retrieve a list of all your InfinityFree hosting accounts.

API Reference

GET /accounts

Retrieve a list of all hosting accounts associated with your InfinityFree account.

Response Example:

{
  "success": true,
  "accounts": [
    {
      "id": "if0_12345678",
      "name": "mywebsite"
    }
  ]
}
GET /accounts/:accountId/domains

Get all domains associated with a specific hosting account.

Example Request:

GET /accounts/if0_12345678/domains

Response Example:

{
  "success": true,
  "domains": [
    "mysite.infinityfreeapp.com",
    "example.com"
  ]
}
GET /accounts/:accountId/domains/:domain/dns

Retrieve DNS records for a specific domain.

Example Request:

GET /accounts/if0_12345678/domains/example.com/dns

Response Example:

{
  "success": true,
  "records": [
    {
      "domain": "example.com",
      "type": "A",
      "target": "192.0.2.1"
    },
    {
      "domain": "www.example.com",
      "type": "CNAME",
      "target": "example.com"
    }
  ]
}
GET /accounts/:accountId/subdomain-extensions

Get available free subdomain extensions for an account (e.g., .infinityfreeapp.com, .rf.gd).

Response Example:

{
  "success": true,
  "extensions": [
    {
      "value": "infinityfreeapp.com",
      "label": "infinityfreeapp.com"
    },
    {
      "value": "rf.gd",
      "label": "rf.gd"
    }
  ]
}
POST /register-domain

Register a new free InfinityFree subdomain.

Request Body (application/x-www-form-urlencoded or application/json):

{
  "accountId": "if0_12345678",
  "subdomain": "mysite",
  "domainExtension": "infinityfreeapp.com"
}

Response:

Returns HTML page with success or error message
POST /register-subdomain

Register a subdomain under your custom domain that's already added to InfinityFree.

Request Body:

{
  "accountId": "if0_12345678",
  "parentDomain": "example.com",
  "subdomain": "blog"
}

Response:

Returns HTML page with success or error message
Creates: blog.example.com
POST /create-cname

Create a CNAME DNS record for a domain.

Request Body:

{
  "accountId": "if0_12345678",
  "domain": "example.com",
  "host": "www",
  "target": "target.example.com"
}

Response:

Returns HTML page with success or error message
POST /verify-auth

Verify that your InfinityFree authentication cookies are valid and working.

Response:

Returns HTML page with authentication status
